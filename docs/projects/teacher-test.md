# Tea Time

> 초등임용 대비 모바일 학습 앱

*소스: [public/data/projects/teacher-test.json](../../public/data/projects/teacher-test.json) + [public/locales/ko/projects/project-teacher-test.json](../../public/locales/ko/projects/project-teacher-test.json)*

## Overview

| 항목 | 내용 |
|---|---|
| 구분 | 개인 프로젝트 |
| 기간 | 2024.01 ~ 2024.02 |
| 기술 스택 | Flutter, Provider, Material Design, Firebase |

대한민국 초등교사 임용 시험을 준비하는 예비 교사들을 위한 모바일 학습 앱임. 방대한 교육과정 내용을 언제 어디서든 퀴즈 형태로 학습하고, 틀린 문제를 체계적으로 관리하여 합격에 한 걸음 더 다가갈 수 있도록 도움. Flutter로 제작되어 이동 시간이나 자투리 시간에도 핵심 내용을 효율적으로 학습할 수 있도록 설계됨.

**주요 기능**: 교육과정/총론 기반 테스트, 오답노트

## Features

- **교육과정 기반 문제** — 초등 전 학년·전 과목의 교육과정(성취기준) 기반 문제 제공
- **다양한 학습 모드** — 과목별·학년별·랜덤 등 다양한 학습 모드로 실력 점검
- **체계적인 오답노트** — 틀린 문제를 자동으로 기록하고 해설과 함께 재학습 가능
- **학습 통계** — 전체 정답률·과목별 정답률 등을 시각화하여 학습 현황 파악

## Development

### 메인 화면 구현

메인 화면에서는 공부에 대한 의욕을 자극하고자 남은 시험 일수와 테스트 결과 정보를 제공함. 시험일 계산은 11월 1일이 일요일인 경우 오는 7일을 시험일로 정하고, 그렇지 않은 경우 1일의 요일을 참고하여 다음 주 토요일을 계산함. 오답 목록은 테스트에서 틀린 문제의 정보를 Firestore에 저장하는 형태로 관리됨. 테스트가 끝나면 오답 목록과 타임스탬프를 업로드함. 다른 기기에서는 저장된 타임스탬프가 최신이 아닐 경우 데이터를 내려받음.

### 교육과정 테스트 다차원 배열 구현

교육과정 내용체계표 자체를 보여주거나 테스트 가능하게 하기 위해 각 값을 컴포넌트화함. 또한 학년군에 따라 렌더링 처리를 다르게 해야 함. 이를 위해 다차원 배열을 활용함. 해당 배열은 과목 > 학년군 > 영역 > 범주 > 세부내용 차례로 구체화됨. 배열을 구체화시킨 덕분에 함수 구현이 간단해졌고 오답노트 구현 시 자세한 정보를 추가할 수 있었음.

### 정답 비교 기능

문장 수준에서 틀린 부분을 찾는 데 시간이 오래 걸리므로 틀린 부분을 정확하게 알 수 있다면 채점 후 오답 정리 시간이 절감됨. 띄어쓰기 포함/미포함 비교로 나누어 한 글자씩 비교하고, 틀린 부분은 빨간 배경으로 표시되도록 구현함. 오답 확인 시 사용자 경험이 향상됨.
