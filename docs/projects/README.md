## 프로젝트 MD 공통 구조

각 프로젝트 MD는 아래 구조를 따름 (원본 [public/data/projects/project-schema.json](../../public/data/projects/project-schema.json) 기반).

1. **Overview** — 구분, 기간, 한 줄 소개, 주요 기능, 기술 스택, 역할, 링크(GitHub/Demo)
2. **Features** — 주요 기능
3. **Development** — 구현 기능 및 개발 과정
4. **Troubleshooting** — 트러블슈팅
5. **Results** — 결과 및 회고
6. **Development Process** — 단계별 개발 흐름
7. **License**
8. **Claude 협업 정보** (해당 프로젝트에만 표기)
